Detailed Description

Allows the user to create and integrate their own functions into the model By specifying the name in the model, and assigning the ptr in the externalFuncs class. *At any State or transition their function will be called and return float *The float will then be applied in the trellis added to transition or emission. (**Adding in log space) *Stores pointers to user functions used by the State's Emissions and Transitions
